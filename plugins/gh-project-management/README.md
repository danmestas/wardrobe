# GitHub Project Management

Create, configure, and manage GitHub Projects via the gh CLI.

## Skills

| Skill | When to use |
|-------|-------------|
| **gh-project-setup** | Creating new projects, applying templates (kanban, roadmap, etc.) |
| **gh-project-operations** | Adding/updating issues, bulk operations, status changes |
| **gh-project-charter** | Creating project charters, documenting goals and scope |
| **gh-project-shared** | Shared utilities (auth, config, error handling) — loaded by other skills |

## Templates

Includes 6 project templates: kanban, bug-tracker, feature-development, roadmap, research, release-planning.

## Install

```bash
claude plugin marketplace add danmestas/agent-plugins
claude plugin install gh-project-management
```
